# games-dots-docs

Public documentation site for the Dots app.

This repository exists so the application source repository can stay private while GitHub Pages hosts a public privacy policy URL for Google Play.

## Publish model

- Repository visibility: public
- GitHub Pages source: repository root on the default branch
- Main page: `index.html`

## Maintenance

Before using the published URL in production, replace `REPLACE_WITH_YOUR_SUPPORT_EMAIL` in `index.html` with a real support email address.
